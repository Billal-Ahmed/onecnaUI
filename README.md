  # ONECNA.COM Frontend Template
</h1>

Welcome to ONECNA.COM, the Crime and Central News Agency located in Pakistan, Karachi.

## About

ONECNA.COM is a news agency dedicated to providing timely and reliable information about crime and central news in the region. This repository contains the frontend code for our website. The project is designed using Tailwind CSS for a modern and responsive UI.

## Project Structure

- `index.html`: The main HTML file containing the structure of the website.
- `css/index-NS5FlIzl.css`: The CSS file generated by Tailwind CSS to style the website.
- `js/index.js`: Custom JavaScript code for enhancing the website's functionality.

## Technologies Used

- [Tailwind CSS](https://tailwindcss.com/): A utility-first CSS framework used for styling.
- HTML: Markup language for structuring the website.
- JavaScript: Scripting language for adding interactivity.
